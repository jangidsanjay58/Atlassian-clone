Button

Flag

A flag is used for confirmations, alerts, and acknowledgments that using a flag group.

Box

A box is a primitive component generic container, and provides managed access to design tokens.

Inline

Inline is a primitive component based on flexbox that manages the horizontal layout of direct children.

Stack

Stack is a primitive component based on flexbox that manages the vertical layout of direct children.
